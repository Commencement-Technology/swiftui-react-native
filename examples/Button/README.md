# Button

## react-native-swiftui

```jsx
<Button action={doSomething}>
  <Text>Click me</Text>
</Button>
```

or

```jsx
<Button text='Click me' action={doSomething} />
```

## SwiftUI

```swift
  Button(action: doSomething) {
    Text("Click me")
  }
```

or

```swift
  Button("Click me", action: doSomething)
```
